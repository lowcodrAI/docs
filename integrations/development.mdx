## What Kontor Can Do

<Tabs>
  <Tab title="GitHub">
    | Capability | Description |
    |---|---|
    | **Manage repositories** | Browse repos, view files, and check status. |
    | **Issues** | Create, update, and search issues. |
    | **Pull requests** | Review PRs, check CI status, and merge. |
    | **Workflows** | View and trigger GitHub Actions. |
  </Tab>
  <Tab title="GitLab">
    | Capability | Description |
    |---|---|
    | **Manage projects** | Browse projects and view repository contents. |
    | **Merge requests** | Create, review, and manage MRs. |
    | **CI/CD pipelines** | View pipeline status and trigger builds. |
  </Tab>
</Tabs>
