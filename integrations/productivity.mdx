## What Kontor Can Do

<Tabs>
  <Tab title="Notion">
    Create and update pages, query databases, and manage workspace content. Kontor can read your Notion data to build context for tasks and decisions.
  </Tab>
  <Tab title="Google Drive">
    Upload and download files, manage sharing permissions, and organize folders. Useful for storing generated documents and accessing reference files.
  </Tab>
  <Tab title="Google Docs">
    Create new documents and edit existing ones. Kontor can draft proposals, meeting notes, and reports directly in your Google Docs.
  </Tab>
  <Tab title="Google Sheets">
    Read and write spreadsheet data, create new sheets, and perform analysis. Ideal for tracking metrics, managing lists, and generating reports.
  </Tab>
  <Tab title="Airtable">
    Manage bases, create and update records, and query tables. Connect your Airtable workflows to Kontor for automated data management.
  </Tab>
  <Tab title="Google Tasks">
    Create tasks, manage to-do lists, and mark items complete. Kontor can turn action items from meetings and emails into tasks automatically.
  </Tab>
</Tabs>
