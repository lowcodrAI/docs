## What Kontor Can Do

<Tabs>
  <Tab title="Linear">
    | Capability | Description |
    |---|---|
    | **Create issues** | File bugs and feature requests from chat or meetings. |
    | **Track progress** | View issue status and sprint progress. |
    | **Manage sprints** | Organize work into cycles and milestones. |
  </Tab>
  <Tab title="Asana">
    | Capability | Description |
    |---|---|
    | **Create tasks** | Add tasks with descriptions, assignees, and due dates. |
    | **Manage projects** | View and update project timelines and status. |
  </Tab>
  <Tab title="Trello">
    | Capability | Description |
    |---|---|
    | **Manage boards** | Create and organize Trello boards. |
    | **Move cards** | Update card status across lists. |
    | **Add details** | Attach descriptions, labels, and checklists. |
  </Tab>
  <Tab title="Jira">
    | Capability | Description |
    |---|---|
    | **Create issues** | File stories, bugs, and tasks. |
    | **Manage sprints** | Plan and track sprint progress. |
    | **Track epics** | Organize work into larger initiatives. |
  </Tab>
</Tabs>
